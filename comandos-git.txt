Comandos GIT

git init ===  Comando usado para inicializar o git na maquina e transformar o directorio actual em repositorio

git status === Usado para rastrear/ver situacao dos arquivos

git add filename ou . para que execute todos arquivos do directorio

git commit -m "descricao do commit" === Comando usado para adicionar os arquivos ao repositorio local

git log === Comando usado para verificar todos commits realizados.
